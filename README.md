# 🫁 Lung Cancer Detection System

A deep learning–based research system for detecting and classifying lung cancer from CT scan images using a ResNet50-based CNN.  
The model achieves **96.82% accuracy**.

> ⚠️ For research and educational purposes only.

---

## 🚀 Features
- Normal / Benign / Malignant classification  
- Heatmap visualization  
- Risk & stage estimation  
- Gradio UI
